# Гидродинамический симулятор (Hydrodynamic Simulator)

Веб-приложение для гидродинамического моделирования нефтяных и газовых
залежей на основе файлов формата `.DAT` (Eclipse-подобный синтаксис).

## Возможности

- Загрузка `.DAT` файлов моделей с парсингом ключевых слов
  (`SPECGRID`, `INCL`, `COOR`, `ZCOR`, `PORO`, `K_X`, `SWAT` и др.)
- Загрузка include-файлов (`.mgrid`, `.minit`) в директорию модели
- Вычисление геометрии ячеек и объёмов с помощью `ConvexHull`
- Визуализация карт средних значений по Z:
  - Проницаемость (Permeability)
  - Пористость (Porosity)
  - NTG (Net-to-Gross)
  - Водонасыщенность / Нефтенасыщенность
  - Объёмы и плотность подвижных запасов нефти

## Технологии

- **Backend:** Python 3.11+, FastAPI, Starlette, Uvicorn
- **Шаблоны:** Jinja2, Bootstrap 5 (CDN)
- **Вычисления:** NumPy, SciPy (ConvexHull), Matplotlib
- **Файлы:** xtgeo (установлен, в текущем коде не задействован)

## Установка и запуск

```bash
# Активировать виртуальное окружение
source /home/ed_ubuntu/.virtualenvs/fastApiProject/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Запустить сервер
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Приложение будет доступно по адресу [http://localhost:8000](http://localhost:8000).

## Использование

1. Откройте главную страницу (`/`).
2. Загрузите `.DAT` файл модели — будут автоматически распознаны
   параметры сетки и список необходимых include-файлов.
3. Загрузите include-файлы (`.mgrid`, `.minit`) через форму.
4. Нажмите **Calculate** (`/calculate`) — парсинг сетки и
   инициализации, вычисление объёмов ячеек.
5. Перейдите на **Plot** (`/plot`) для просмотра карт средних
   значений.

## Формат входных файлов

- **`.DAT`** — основной файл модели с секциями `SPECGRID`, `INCL` и др.
  Поддерживается нотация повторений (`3*0.25` → `0.25 0.25 0.25`).
- **`.mgrid`** — геометрия сетки (COOR, ZCOR).
- **`.minit`** — инициализация (PORO, PERMX, SWAT, etc.).

## Примечания

- Состояние модели хранится в глобальном словаре `MODEL_STATE`
  и **сбрасывается при перезапуске сервера**.
- `xtgeo` импортирован, но в текущей версии кода не используется.
- Тесты отсутствуют — тестирование вручную через браузер.
